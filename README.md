# aws_get_session_token
Getting session token

Your file with aws credentials should be like:
```
[profile1]
aws_access_key_id = xxxx
aws_secret_access_key = xxxxxxxxxxxxxxxxxxxxxx

[profile2]
aws_access_key_id = xxxx
aws_secret_access_key = xxxxxxxxxxxxxxxxxxxxxx

....

[default]
aws_access_key_id = xxxx
aws_secret_access_key = xxxxxxxxxxxxxxxxxxxxxx
```
